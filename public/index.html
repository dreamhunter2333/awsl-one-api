🚀 欢迎使用 AWSL One API

AWSL One API 是一个强大的 API 代理和管理平台，为您提供统一的 AI 模型接口管理服务。

🔗 频道管理

配置和管理 AI 服务提供商的 API 端点，支持 Azure OpenAI 等多种平台。

🔑 令牌管理

生成和管理 API 访问令牌，控制用户访问权限和使用配额。

💰 定价管理

灵活配置不同 AI 模型的计费策略，精确控制使用成本。

🧪 API 测试

内置 API 测试工具，快速验证配置和调试接口问题。

⚡ 快速开始

  1. 首先初始化数据库以设置必要的数据表结构
  2. 在频道管理中添加您的 AI 服务提供商配置
  3. 创建访问令牌并设置适当的权限和配额
  4. 配置模型定价策略以控制使用成本
  5. 使用 API 测试工具验证配置是否正常工作

定价配置

💡 定价说明

基础配额单位：1百万token = $1.00 | 定价配置为不同模型相对于基础单位的倍率 | 例如：gpt-4的input倍率为30，表示1百万token消耗30美元配额

模型名称 输入倍率 输出倍率 操作
配置格式：模型名称 -> {input: 输入倍率, output: 输出倍率}。倍率基于1百万token=$1.00计算
